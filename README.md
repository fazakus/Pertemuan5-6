***Source code*** untuk aplikasi Tugas 5 ([Tugas 5 Python](p5_tugas.py)) :
```python
print("Please enter full name : ")
fullname=input()

print("please insert your Nickname : ")
nickname=input()

bornplace=input()

print("please insert your age : ")
age=input()

print("please enter your home address : ")
address=input()
```
